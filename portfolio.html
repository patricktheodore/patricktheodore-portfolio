Renters Review

Renters Review is an intuitive application that allows users to view and add reviews on rental properties on the market. Allowing them to save time of going to viewings and potentially protect themselves from hidden issues within properties. The application runs without login credentials to an extent, but for a User to add properties and/or reviews, they must create an account and be logged in to do so. User's sensitive information like passwords, are encrypted using bcrypt.

Technologies Used: bcrypt, dotenv, express, handlebars, mysql2, sequelize, eslint, prettier, heroku
